# Syntecxhub_Project_python

Simple Python Calculator

## Overview
A beginner-friendly command-line calculator built using Python. It supports arithmetic operations (addition, subtraction, multiplication, division, modulus, exponent) and math functions from Python's `math` module. The tool validates input, handles errors like divide-by-zero, and keeps a small history of calculations.

## Features
- Evaluate mathematical expressions, e.g. `2 + 3 * (4 - 1)`
- Use math functions: `sqrt(9)`, `sin(0.5)`, etc.
- Commands:
  - `:history` — show past calculations
  - `:clear` — clear history
  - `:help` — show help message
  - `:exit` — quit the app
- Safer `eval` via a controlled environment
